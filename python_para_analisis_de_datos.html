No. El curso arranca desde las bases estructurales del pensamiento lógico y los comandos más elementales de Python. Está diseñado precisamente para perfiles que se inician desde cero.

Sí, todas las sesiones se almacenan en alta definición para que puedas repasar los códigos, ejercicios y explicaciones teóricas a tu propio ritmo cuantas veces lo requieras.

Utilizaremos Python y el entorno de Jupyter Notebook. Durante la primera sesión te guiaremos paso a paso en la instalación y configuración de estas herramientas en tu computadora (Windows, Mac o Linux).
