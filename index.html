Ver programa

Un programa diseñado para introducirte en el mundo del análisis y visualización de datos. Aprende paso a paso las herramientas y el pensamiento analítico que utilizan los profesionales en ciencia de datos.

TEOCALLI LAB

Transforma datos en conocimiento. Aprende las herramientas y técnicas esenciales para el análisis profesional de datos.

01 Pensamiento estadístico +

Desarrolla tu capacidad analítica con fundamentos estadísticos sólidos. Aprende a interpretar datos, identificar patrones y tomar decisiones basadas en evidencia cuantitativa.

02 Excel +

Domina Excel más allá de lo básico. Fórmulas avanzadas, tablas dinámicas, visualizaciones y automatización con macros para análisis de datos profesional.

03 Base de Datos SQL +

Aprende a consultar, manipular y gestionar bases de datos relacionales. Desde queries básicos hasta optimización de consultas complejas.

04 Tableau +

Crea visualizaciones de datos impactantes e interactivas. Aprende a contar historias con datos y crear dashboards profesionales.

05 Power BI +

Domina la herramienta de Business Intelligence de Microsoft. Conecta múltiples fuentes de datos, modela información y crea reportes dinámicos.

06 R Studio +

Programa en R para análisis estadístico avanzado. Manipulación de datos, modelado predictivo y visualizaciones con ggplot2.

07 Python +

Aprende Python para ciencia de datos. Pandas, NumPy, análisis exploratorio, machine learning y automatización de procesos analíticos.
